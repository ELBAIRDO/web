config.resource_monitor.fixed_heap.v2alpha.FixedHeapConfig
----------------------------------------------------------

`[config.resource_monitor.fixed_heap.v2alpha.FixedHeapConfig proto] <https://github.com/envoyproxy/envoy/blob/5867e33124deac887c7100f9c3b3480e1fd67f42/api/envoy/config/resource_monitor/fixed_heap/v2alpha/fixed_heap.proto#L14>`_

The fixed heap resource monitor reports the Envoy process memory pressure, computed as a
fraction of currently reserved heap memory divided by a statically configured maximum
specified in the FixedHeapConfig.

    "max_heap_size_bytes": "..."
