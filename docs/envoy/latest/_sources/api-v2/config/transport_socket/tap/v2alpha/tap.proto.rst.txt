config.transport_socket.tap.v2alpha.Tap
---------------------------------------

`[config.transport_socket.tap.v2alpha.Tap proto] <https://github.com/envoyproxy/envoy/blob/36cf26b9d6ac32f5671c24b92f9ea22ab70bc6c1/api/envoy/config/transport_socket/tap/v2alpha/tap.proto#L17>`_

Configuration for tap transport socket. This wraps another transport socket, providing the
ability to interpose and record in plain text any traffic that is surfaced to Envoy.

.. code-block:: json

  {
    "common_config": "{...}",
    "transport_socket": "{...}"
  }
