config.listener.v2.ApiListener
------------------------------

`[config.listener.v2.ApiListener proto] <https://github.com/envoyproxy/envoy/blob/424909395c90d7d68f1afeb3427c26c7c85f2672/api/envoy/config/listener/v2/api_listener.proto#L19>`_

Describes a type of API listener, which is used in non-proxy clients. The type of API
exposed to the non-proxy application depends on the type of API listener.

.. code-block:: json

  {
    "api_listener": "{...}"
  }

.. _envoy_api_field_config.listener.v2.ApiListener.api_listener:

api_listener
  (`Any <https://developers.google.com/protocol-buffers/docs/reference/google.protobuf#any>`_) The type in this field determines the type of API listener. At present, the following
  types are supported:
  envoy.config.filter.network.http_connection_manager.v2.HttpConnectionManager (HTTP)
