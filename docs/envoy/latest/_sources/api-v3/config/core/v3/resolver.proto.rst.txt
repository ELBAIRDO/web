config.core.v3.DnsResolver
--------------------------

:repo:`[config.core.v3.DnsResolver proto] <api/envoy/config/core/v3/resolver.proto#L17>`

DNS resolver configuration which includes the underlying dns resolver addresses and options.

.. code-block:: json

  {
    "resolvers": []
  }

.. _envoy_v3_api_field_config.core.v3.DnsResolver.resolvers:

resolvers
  (**repeated** :ref:`config.core.v3.Address <envoy_v3_api_msg_config.core.v3.Address>`, *REQUIRED*) A list of dns resolver addresses
  Setting this value causes failure if the
  ``envoy.restart_features.use_apple_api_for_dns_lookups`` runtime value is true during
  server startup. Apple's API only allows overriding DNS resolvers via system settings.
