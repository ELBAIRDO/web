.. _envoy_v3_api_field_config.listener.v3.UdpListenerConfig.max_downstream_rx_datagram_size:

max_downstream_rx_datagram_size
  (`UInt64Value <https://developers.google.com/protocol-buffers/docs/reference/google.protobuf#uint64value>`_) The maximum size of received downstream UDP datagrams. Using a larger size will cause Envoy to allocate
  more memory per listener. Received datagrams above this size will be dropped. If not set
  defaults to 1500 bytes.
