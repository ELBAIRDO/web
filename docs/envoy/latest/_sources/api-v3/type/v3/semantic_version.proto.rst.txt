type.v3.SemanticVersion
-----------------------

`[type.v3.SemanticVersion proto] <https://github.com/envoyproxy/envoy/blob/ce3fe2e9d04f7d40fb740b9c0ab9d047edc23b55/api/envoy/type/v3/semantic_version.proto#L17>`_

Envoy uses SemVer (https://semver.org/). Major/minor versions indicate
expected behaviors and APIs, the patch version field is used only
for security fixes and can be generally ignored.

.. code-block:: json

  {
    "major_number": "...",
    "minor_number": "...",
    "patch": "..."
  }
