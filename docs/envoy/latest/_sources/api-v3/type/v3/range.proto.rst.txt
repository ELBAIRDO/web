type.v3.Int64Range
------------------

`[type.v3.Int64Range proto] <https://github.com/envoyproxy/envoy/blob/28d9426cc190e5d4b4d7604d81754825fbc18178/api/envoy/type/v3/range.proto#L14>`_

Specifies the int64 start and end of the range using half-open interval semantics [start,
end).

  {
    "start": "...",
    "end": "..."
  }
